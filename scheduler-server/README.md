## 开发

### 开发环境

 * python v3.8.10
 * pip(python-pip3) v20.0.2 
 * flask v2.2.2
 * docker v6.0.0
 * flask-cors v3.0.10
 * tornado v6.2

### 开发运行

1. 在`terminal`中，切换目录到`scheduler-server`下。
2. 输入`sudo python tornado_server.py`运行项目。
