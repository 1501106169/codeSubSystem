## Development

### Development Environment

 * python v3.8.10
 * pip(python-pip3) v20.0.2 
 * flask v2.2.2
 * docker v6.0.0
 * flask-cors v3.0.10
 * tornado v6.2

### Running

1. In `terminal`, switch the directory to `scheduler-server`.
2. Enter `sudo python tornado_server.py` to run the project.
